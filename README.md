# Advanced Machine Learning Assignment

## Overview

This repository contains implementations of basic Machine Learning algorithms using Python and Scikit-learn.

## Topics Covered

* Decision Tree vs Random Forest
* Random Forest Hyperparameter Tuning
* Support Vector Machine (SVM)
* Effect of C parameter in SVM
* KNN (Euclidean vs Manhattan)
* Feature Scaling in KNN
* PCA (Principal Component Analysis)
* Confusion Matrix, Precision, Recall, F1-score
* K-Fold Cross Validation
* KNN with different K values
* Bagging vs Single Model

## Tools Used

* Python
* Scikit-learn
* Matplotlib
* Google Colab
